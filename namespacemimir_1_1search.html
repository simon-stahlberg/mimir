Return true iff the conditional effect is applicable in the problem and state when it fires, i.e., when the conjunctive condition is applicable then the effect must also be applicable. More formally, consider conditional effect c = <pre, eff>. app(c) <=> app(pre) -> app(eff) <=> !app(pre) || app(eff) <=> !(app(pre) && !app(eff)) <=> !(!app(eff) && app(pre)) Now we have a more efficient form that tests app(eff) before app(pre).
