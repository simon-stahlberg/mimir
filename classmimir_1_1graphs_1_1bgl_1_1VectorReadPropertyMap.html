Detailed Description

template<std::unsigned_integral I, typename Value>
class mimir::graphs::bgl::VectorReadPropertyMap< I, Value >

VectorReadPropertyMap provides read-access to the Value associated with a given Key via a call to the get() function for dense graph structures whose underlying storage are vectors.
