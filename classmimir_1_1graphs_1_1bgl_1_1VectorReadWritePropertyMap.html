Detailed Description

template<std::unsigned_integral I, typename Value>
class mimir::graphs::bgl::VectorReadWritePropertyMap< I, Value >

VectorReadWritePropertyMap provides read-write-access to the Value associated with a given Key via a call to the get() and put() function for dense graph structures whose underlying storage are vectors.

Link to Boost docs: https://www.boost.org/doc/libs/1_82_0/libs/property_map/doc/ReadWritePropertyMap.html
