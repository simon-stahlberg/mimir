Detailed Description

template<IsStaticOrFluentTag F>
class mimir::formalism::NumericAssignmentSet< F >

NumericAssignmentSet is a helper class representing a set of functions f : FunctionSkeleton x Params(A) x Object x Params(A) x Object -> Bounds<ContinuousCost> where

  1. f(p,i,o,j,o') = [l,r] iff min_{...} h(...,o_i,...,o'j,...) = l and max{...} h(...,o_i,...,o'j,...) = r
  2. f(p,i,o,-1,-1) = [l,r] iff min{...} h(...,o_i,...) = l and max_{...} h(...,o_i,...) = r
  3. for all others, we have f(p,i,o,j,o') = [inf,-inf] and f(p,i,o,-1,-1) = [inf,-inf] with respective meanings
  1. the assignment [i/o], [j/o'] results in partial function evaluation [l,r]
  2. the assignment [i/o] results in partial function evaluation [l,r] Using the NumericAssignmentSet we can efficiently evaluate numeric constraints partially.
