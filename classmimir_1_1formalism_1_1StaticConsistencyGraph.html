Detailed Description

The StaticConsistencyGraph encodes the assignments to static conditions, and hence, it is an overapproximation of the actual consistency graph.

Construct a static consistency graph @parameter_index problem The problem context. @parameter_index begin_parameter_index The first parameter index for which consistent assignments are represented. @parameter_index end_parameter_index The last parameter index plus one for which consistent assignments are represented. @parameter_index static_conditions The static literals for which a bindings must be found. @parameter_index static_assignment_set The assignment set of static initial literals.

For universal effects, we can set first and last parameter indices respectively to find consistent assignments irrespective of the action parameter bindings. More specifically:

  1. for action parameters, we set first to 0 and last to 0 + arity(action)
  2. for universal effects, we set first to arity(action) and last to arity(action) + arity(effect)
