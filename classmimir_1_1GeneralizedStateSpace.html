Detailed Description

GeneralizedStateSpace encapsulates a ProblemGraphList Q with an additional ClassStateSpace structure on top.

GeneralizedStateSpace

There is a one-to-many mapping from ClassVertex (resp. ClassEdge) to ProblemVertex (resp. ProblemEdge), which is a strict one-to-many mapping, if symmetries across different problems are detected. From each ClassVertex (resp. ClassEdge) one can access the single representative ProblemVertex (resp. ProblemEdge). From each ProblemVertex (resp. ProblemEdge) one can access the corresponding ClassVertex (resp. ClassEdge).
